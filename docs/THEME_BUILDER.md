This document serves as a specification for creating new themes for **Core CMS**. Use this context when asking AI to generate a new theme.

## 1. Directory Structure
Create a new folder in `themes/` (e.g., `themes/my-new-theme/`).

**Required Files:**
*   `header.php`
*   `footer.php`
*   `404.php`
*   `style.css` (or `assets/css/style.css`)

## 2. Required Files & Logic

### `header.php`
Must handle the HTML head and opening body tags.

**Checklist:**
*   [ ] **DocType**: `<!DOCTYPE html>`
*   [ ] **Meta**: Charset, Viewport.
*   [ ] **Title**: Dynamic title logic:
    ```php
    <title><?php echo isset($page_title) ? htmlspecialchars($page_title) . ' - ' . htmlspecialchars(get_option('site_title')) : htmlspecialchars(get_option('site_title')); ?></title>
    ```
*   [ ] **Styles**: Link to your theme's CSS using `BASE_URL`.
    ```php
    <link rel="stylesheet" href="<?php echo BASE_URL; ?>/themes/my-new-theme/style.css">
    ```
*   [ ] **Site Identity**: Display Logo or Title.
    ```php
    $logo = get_option('site_logo');
    if ($logo) { echo "<img src='$logo'>"; }
    ```
*   [ ] **Navigation**: Render the main menu.
    ```php
    render_menu('main-menu', 'your-menu-class');
    ```
*   [ ] **Breadcrumbs**: (Optional but recommended)
    ```php
    if (function_exists('render_breadcrumbs')) render_breadcrumbs($page_title ?? '');
    ```

### `footer.php`
Must handle the closing tags.

**Checklist:**
*   [ ] **Footer Text**: Display the dynamic footer text.
    ```php
    echo get_option('site_footer_text', '&copy; ' . date('Y'));
    ```
*   [ ] **Scroll to Top**: (Optional) Include logic if supported.
*   [ ] **Closing Tags**: `</body>`, `</html>`.

### `404.php`
A user-friendly "Page Not Found" layout.

## 3. CSS Standards
To ensure compatibility with Core CMS features (like the Block Editor), your CSS should support these classes:

### Layout
*   `.container`: Centered content wrapper.
*   `.site-header`, `.site-main`, `.site-footer`: Semantic regions.

### Navigation
*   `.site-menu` (or your custom class): The `<ul>` generated by `render_menu()`.
*   `.site-menu ul`: Sub-menus (Dropdowns).

### Editor Content (Inside `.entry-content`)
*   `h1`, `h2`, `h3`: Headings.
*   `p`, `ul`, `ol`, `li`: Standard text.
*   `blockquote`, `.editor-quote`: Quotes.
*   `.editor-image`: Images with captions.
*   `.editor-delimiter`: Horizontal rules/separators.
*   `pre`, `code`: Code blocks (Monospace font, dark background recommended).

## 4. Helper Functions
*   `get_option($key, $default)`: Retrieve settings.
*   `get_label($key, $default)`: Retrieve localized UI text.
*   `render_menu($slug, $class)`: Output navigation.
*   `render_breadcrumbs($title)`: Output breadcrumbs.
